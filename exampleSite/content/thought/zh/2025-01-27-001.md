今天完成了说说页面的开发，使用了 Hugo 静态站点生成器和 Tailwind CSS 框架。
```shell
cd seven/exampleSite

hugo mod npm pack

npm install
```
#### 主要功能

1. **热力图展示** - 类似 GitHub 的活跃度展示
2. **时间线布局** - 按时间顺序展示说说
3. **支持配图** - 可以附加多张图片
4. **国际化支持** - 中英文切换
